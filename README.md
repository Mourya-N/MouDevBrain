# MouCodeBrain - AI That Understands Any Codebase

A powerful tool where developers can chat with their entire codebase using natural language.

## 🚀 Features

- **Codebase Chat**: Ask questions about your project in natural language
- **Architecture Visualization**: Auto-generated system architecture diagrams
- **Dependency Mapping**: Visualize relationships between files and modules
- **Bug Finder**: AI-powered code analysis for bugs, security issues, and bad patterns
- **VS Code Extension**: Context-aware assistance directly in your editor

## 🛠️ Tech Stack

- **Frontend**: React + Tailwind CSS
- **Backend**: Spring Boot, JWT Authentication
- **AI Engine**: Python FastAPI, LangChain, OpenAI
- **Databases**: MongoDB Atlas, FAISS Vector DB
- **DevOps**: Docker, Docker Compose

## 📋 Prerequisites

- Docker and Docker Compose
- Node.js 18+ (for local development)
- Java 17+ (for local development)
- Python 3.11+ (for local development)
- OpenAI API Key
